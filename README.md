# 3.6.2.Programming_language_Prolog
# Eglītes zīmēšana no trijstūriem 

Šī Prolog programma izvada ekrānā eglītes attēlu, kas ir veidota no vairākiem trijstūriem. <br>
<br>
Katrs trijstūris tiek zīmēts no simboliem `*`, un tā izmēru nosaka trijstūra malas garums (augstums), kas ir vesels skaitlis.<br>
<br>
Programmai jāietver predikāts `main/1` ar vienu argumentu — veselu skaitļu sarakstu, kur katrs elements apraksta viena trijstūra izmēru.<br>
<br>
Izpildot `main/1`, programma secīgi uzzīmē trijstūrus atbilstoši sarakstā norādītajiem izmēriem, izveidojot eglītes attēlu.<br>
<br>
Katrs trijstūris tiek izvadīts pa rindām, veidojot pakāpeniski pieaugošu `*` skaitu katrā nākamajā rindā.
